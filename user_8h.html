◆ LDUserNew()

struct LDUser* LDUserNew ( const char *const  key)

Allocate a new user.

The user may be modified until it is passed to the LDClientIdentify or LDClientInit. The key argument is not required. When key is NULL then a device specific ID is used. If a device specific ID cannot be obtained then a random fallback is generated.
